Paramorama 2 - Download

Paramorama 2 was implemented in Java. To run the program you will need to have Java installed on your system:

  • To test if you already have Java installed, follow these instructions;
  • If you do not have Java installed, it can be downloaded here by choosing the version of Java suited for your operating system. Follow the provided installation instructions.

To install Paramorama 2 follow these instructions:

  • Download Paramorama2.zip;
  • Unzip this archive to your favourite directory. You should see four files: paramorama2.bat, paramorama2.command, paramorama2.sh, and Paramorama2.jar;
  • If you are using Linux, navigate to the relevant directory and run $ sh paramorama2.sh; or
  • If you are using a Mac double-click on paramorama2.command; or
  • If you are using a Windows machine, double click on paramorama2.bat.

Note: you can also run Paramorama 2 by double clicking on Paramorama2.jar but running the relevant scripts ensures that more memory is allocated to the application than the default. This is often desirable since Paramorama 2 works with image files.
