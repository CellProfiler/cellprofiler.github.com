Using The File Menu

The File menu provides options for loading and saving your pipelines and performing an analysis run.
  • Load Pipeline...: Load a CellProfiler pipeline (.cp or .mat) file from your hard drive.
  • Load Pipeline from URL: Load a CellProfiler pipeline from a web address (i.e., http://...)
  • Save Pipeline: Save the pipeline you are currently working on as a CellProfiler .cp pipeline file.
  • Save Pipeline as...: Prompts the user for a filename to save the pipeline as.
  • Clear Pipeline: Removes all modules from the current pipeline.
  • Open Image: Opens a dialog box prompting you to select an image file for display. Images listed in the file panel can be also be displayed by double-clicking on the filename.
  • Analyze Images: Executes the current pipeline using the current module and Default Input and Output folder settings.
  • Stop Analysis: Stop the current analysis run.
  • Run Multiple Pipelines: Execute multiple pipelines in sequential order. This option opens a dialog box allowing you to select the pipelines you would like to run as well as the associated input and output folders. See the help in the Run Multiple Pipelines dialog for more details.
  • Restart: Resume a partially completed analysis run from here it left off. To restart an analysis run, go to File > Restart choose the output .mat file containing the partially complete measurements and the analysis run will pick up starting with the last cycle that was processed.
  • Preferences...: Displays the Preferences window, where you can change many options in CellProfiler.
  • Check for updates...: Displays a dialog which lets you know whether your installation of CellProfiler is the latest version and if not, notifies you as what version is available from CellProfiler website.
  • Recent: Displays a list of the most recent pipelines loaded. Select any one of these pipelines to load it.
  • Exit: End the current CellProfiler session. You will be given the option of saving your current pipeline if you have not done so.
