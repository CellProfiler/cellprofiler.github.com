Using The Edit Menu

The Edit menu provides options for modifying modules in your current pipeline.
  • Undo: Undo your previous module modification.
  • Move Up: Move the currently selected module(s) up in the module list. You can also use the button located below the Pipeline panel.
  • Move Down: Move the currently selected module(s) down in the module list. You can also use the button located below the Pipeline panel.
  • Delete: Remove the currently selected module(s) from the module list. Pressing the Delete key also removes the module(s). You can also use the button located under the Pipeline panel.
  • Duplicate: Duplicate the currently selected module(s) in the pipeline. The current settings of the selected module(s) are retained in the duplicate.
  • Add Module: Select a module from the pop-up list to inster into the current pipeline. You can also use the button located under the Pipeline panel.
You can select multiple modules at once for moving, deletion and duplication by selecting the first module and using Shift-click on the last module to select all the modules in between.
